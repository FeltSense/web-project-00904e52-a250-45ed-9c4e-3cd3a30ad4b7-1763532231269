# ShopVibe - Social Shopping Platform

## Project Overview
ShopVibe is a modern retail business website designed for general consumers, featuring a comprehensive social shopping experience with rating systems, trending content discovery, and community-driven product recommendations.

## Brand Identity
- **Industry**: Retail
- **Target Audience**: General consumers interested in social shopping and product discovery
- **Style**: Modern and professional with vibrant, engaging elements
- **Color Palette**: Blue and purple gradients with clean whites and grays

## Key Features

### 1. Rating/Voting System
- Community-driven product ratings
- Verified purchase reviews
- Star rating displays
- Vote up/down functionality

### 2. Social Discovery
- Follow other shoppers
- Share product discoveries
- Community features
- Social shopping feeds

### 3. Trending Content
- Real-time trending products
- Popular items showcase
- Trending metrics and statistics
- Dynamic content updates

### 4. Download/Sign Up
- Easy registration process
- Mobile app mentions
- Premium membership options
- Account creation flow

### 5. Pricing
- Clear $29/month premium membership
- Stripe integration for payments
- Feature comparison
- Transparent pricing structure

### 6. Testimonials
- Customer success stories
- Verified user reviews
- Star ratings and feedback
- Community testimonials

### 7. Team Profiles
- Implied through testimonials section
- Community-focused approach
- User-centric design

## Technical Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript
- **Payment**: Stripe
- **Forms**: Custom API integration

## Design Philosophy
- Clean, modern interface
- Generous white space
- Professional imagery from Unsplash
- Smooth transitions and hover effects
- Mobile-first responsive design
- Accessible and user-friendly

## Color Scheme
- Primary Blue: #2563eb to #3b82f6
- Secondary Purple: #7c3aed to #a855f7
- Neutral Grays: #f9fafb to #111827
- Accent Colors: Yellow (#fbbf24), Green (#10b981)

## Typography
- Headings: Poppins (Bold, 600-900 weights)
- Body: Inter (Regular, 400-600 weights)
- Large hero text: 6xl to 7xl
- Body text: base to xl

## Page Sections
1. **Navigation**: Sticky header with smooth scroll
2. **Hero**: Full-screen with compelling value proposition
3. **Services/Features**: 6 key features in grid layout
4. **Testimonials**: 6 customer reviews with photos
5. **Pricing**: Single premium tier at $29/month
6. **Contact Form**: Integrated with API endpoint
7. **Footer**: Comprehensive links and information
